# Node Qredit QAE API Helper

A Promised NodeJS Module for connecting with the Qredit QAE API

```
git clone https://github.com/mrmikeo/nodeqaeapi
cd nodeqaeapi
npm install

node index.js
```

const qaeApi = require("./lib/qaeApi");
const qae = new qaeApi.default();
